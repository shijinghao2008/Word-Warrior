把你上传的 5 组像素精灵图放到这个目录下（保持文件名一致），底部导航会自动使用它们：

- `vocab.png` / `vocab_active.png`
- `scholar.png` / `scholar_active.png`
- `arena.png` / `arena_active.png` / `arena_pressed.png`（按下态可选，但推荐提供）
- `leaderboard.png` / `leaderboard_active.png`
- `profile.png` / `profile_active.png`

说明：
- `*_active.png` 用于当前选中/高亮（例如你图里的蓝色发光版本）
- `*_pressed.png` 仅用于“对战”按钮按下效果（你图里的 Normal / Pressed）
- 图片建议带透明背景，并且是像素风（组件已启用 `image-rendering: pixelated`）
